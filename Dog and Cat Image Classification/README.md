# Dog and Cat Image Classification

This project builds a **Deep Learning model to classify images of dogs and cats** using a **Convolutional Neural Network (CNN)**.
The model is implemented using **TensorFlow and Keras** and learns visual features from image data to distinguish between the two classes.

The program loads the image dataset, trains the CNN model, and predicts whether the given image belongs to a **dog or a cat**.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
