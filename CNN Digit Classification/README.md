# CNN Digit Classification

This project implements a **Convolutional Neural Network (CNN) to classify handwritten digits**.
The model is trained on the **MNIST dataset** and built using **TensorFlow and Keras**.

The CNN model learns important image features through convolution and pooling layers to accurately recognize digits from **0 to 9**.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
