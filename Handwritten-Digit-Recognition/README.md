# Handwritten Digit Recognition

This project implements a **Handwritten Digit Recognition system** using Deep Learning.
The model is trained on the **MNIST dataset**, which contains images of handwritten digits from **0 to 9**.

A neural network built with **TensorFlow and Keras** is used to learn patterns from the dataset and classify digits accurately.

The program loads the MNIST dataset, trains the model, and predicts the digit from a given input image.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* OpenCV
