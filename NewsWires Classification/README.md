# Newswire Topic Classification

This project implements a **Deep Learning model to classify news articles into different topics**.
The model is trained using the **Reuters newswire dataset** and built with **TensorFlow and Keras**.

The program preprocesses the news text data, trains a neural network model, and predicts the category of each news article based on its content.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
