# House Price Prediction

This project builds a **Deep Learning model to predict house prices** based on different features such as location, size, and other housing attributes.
The model is trained using the **Boston Housing dataset** and implemented with **TensorFlow and Keras**.

The program preprocesses the dataset, trains a neural network model, and predicts house prices based on the input features.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
