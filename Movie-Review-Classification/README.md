# Movie Review Sentiment Classification

This project builds a **Deep Learning model to classify movie reviews as positive or negative**.
The model is trained using the **IMDB movie review dataset** and implemented with **TensorFlow and Keras**.

The program preprocesses the text data, trains a neural network model, and predicts the sentiment of movie reviews based on the learned patterns.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
