# CNN Image Classification

This project implements **Image Classification using a Convolutional Neural Network (CNN)**.
The model is built with **TensorFlow and Keras** and is trained to recognize and classify images into different categories.

The program processes image data, trains the CNN model, and predicts the class of input images based on learned visual features.

**Technologies Used:**

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
