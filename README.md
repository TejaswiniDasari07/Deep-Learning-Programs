# Deep Learning Programs

This repository contains a collection of **Deep Learning programs implemented using Python and TensorFlow/Keras**.
The programs demonstrate various deep learning concepts including neural networks, convolutional neural networks (CNN), recurrent neural networks (RNN), and natural language processing techniques.

## Programs Included

1. **Handwritten Digit Recognition**
   Implementation of a neural network to recognize handwritten digits using the MNIST dataset.

2. **Movie Review Sentiment Classification**
   Deep learning model to classify movie reviews as positive or negative using the IMDB dataset.

3. **Newswire Topic Classification**
   Model that categorizes news articles into different topics using the Reuters dataset.

4. **House Price Prediction**
   Neural network model used to predict house prices based on different housing features.

5. **CNN Digit Classification**
   Convolutional Neural Network used for handwritten digit recognition.

6. **Dog and Cat Image Classification**
   CNN-based model that classifies images of dogs and cats.

7. **CNN Image Classification**
   Implementation of image classification using convolutional neural networks.

8. **One Hot Encoding**
   Demonstration of converting categorical or text data into numerical format using one-hot encoding.

9. **Word Embeddings**
   Implementation of word embedding techniques for representing words as dense vectors.

10. **Recurrent Neural Network (RNN)**
    Implementation of an RNN model for handling sequential data such as text.

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* OpenCV
* Jupyter Notebook

## Purpose of the Repository

The purpose of this repository is to demonstrate practical implementations of **Deep Learning concepts and techniques** .
